# Vercel 靜態部署修正版

本版本已移除 Vite、package.json 與 npm 套件安裝需求。

Vercel 不應再執行 npm install，也不需要 Build Command。

## 建議 Vercel 專案設定

- Framework Preset：Other
- Build Command：留空
- Output Directory：留空
- Install Command：留空
- Root Directory：專案根目錄
