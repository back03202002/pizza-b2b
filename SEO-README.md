## 上線後請記得修改正式網址

目前預設網址是：

https://pizza-b2b.com/

部署完成後，若你的 Vercel 網址不同，請在 `index.html`、`public/robots.txt`、`public/sitemap.xml` 裡把網址改成正式網址。

## 建議正式上線後做

1. 到 Google Search Console 新增網站
2. 提交 sitemap.xml
3. 檢查行動版可用性
4. 檢查頁面速度
5. 後續增加文章或 landing page，例如：
   - 冷凍披薩批發合作
   - 團爸團媽披薩團購方案
   - 餐飲店冷凍披薩進貨推薦
